## API Documentation
```
POST /api/auth/register {email, password}
POST /api/auth/login {email, password}
GET /api/tasks?page=1&status=Done&priority=High&search=test
POST /api/tasks {title, description, status, priority, dueDate}
PUT /api/tasks/:id {...task}
DELETE /api/tasks/:id
GET /api/analytics → Charts data
```
